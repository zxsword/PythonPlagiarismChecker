# Python 代码相似度检查工具 (Python Plagiarism Checker)

这是一个基于 Python 标准库开发的轻量级、带图形界面的代码查重工具。它可以帮助教师、助教或开发者快速批量比较多个 Python 代码文件，找出潜在的抄袭行为。

## ✨ 核心特性

- **图形化界面 (GUI)**：使用 Tkinter 构建，操作直观，支持拖拽和批量导入文件夹。
- **双重查重模式**：
  - **常规模式**：忽略注释和空行，比较代码的文本相似度。
  - **深度模式**：基于抽象语法树 (AST)，忽略变量重命名作弊，直接比对代码骨架。
- **智能分组引擎**：使用图论算法（深度优先搜索 DFS），自动将互相抄袭的多个文件归为一组。
- **原创源头追踪**：通过启发式规则（创建时间、代码长度等）预测组内谁是“原创者”。
- **多文件对比视图**：专业 IDE 级别的对比窗口，支持代码同步高亮滚动，提供“专业模式”和“查重标红”双主题。
- **一键导出报告**：支持将查重结果导出为 CSV 格式，方便用 Excel 进一步处理。

## 🚀 如何运行

本项目**纯天然无依赖**，仅使用了 Python 的内置标准库，无需使用 `pip` 安装任何第三方包。

### 前提条件
- 需要安装 Python 3.6 或更高版本。

### 运行步骤
1. 下载或克隆本仓库到本地。
2. 双击运行目录下的 `run.bat` 文件（仅限 Windows）。
3. 或者在命令行/终端中运行以下命令：
   ```bash
   python main.py
   ```

## 📂 项目结构

- `main.py`：程序的启动入口。
- `plagiarism_checker/`：核心代码包。
  - `analysis.py`：查重算法核心（AST解析、差异比对、图论算法）。
  - `ui/app.py`：主窗口界面实现。
  - `ui/comparison_window.py`：代码对比高亮窗口实现。
- `tests/`：单元测试目录。
- `run.bat`：Windows 快速启动脚本。
